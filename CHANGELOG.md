### ✨ 一句话概括

### 🎯 产品定位

FadeMemo 是一款**反传统**的笔记 App：不主打「永久存储」，而是模拟人类记忆的自然遗忘规律。

- **核心受众**：愿意和记忆和解、享受文字氛围感的人
- **使用场景**：日记、灵感速记、记忆练习、写作辅助、文艺向工具
- **核心情绪**：「安静 / 留白 / 不强求」

---

### 📐 核心算法（艾宾浩斯遗忘曲线）

```
strength(t) = importance × exp(-λ × t^β)
```

| 参数 | 含义 | 默认值 |
| :--- | :--- | :---: |
| `strength(t)` | 当前记忆强度 0~1 | — |
| `importance` | 字符重要性 0~1（决定初始强度 & 衰减速度） | `0.85` |
| `λ` | 全局遗忘系数（用户滑块调节） | `8e-5` |
| `β` | 衰减曲率，与 importance 反相关 | `0.5 ~ 2.0` |
| `t` | 字符创建后经过的秒数 | — |

**β 派生公式**：`β = 0.5 + (1 - importance) × 1.5`
- importance = 1.0 → β = 0.5（最稳）
- importance = 0.0 → β = 2.0（最快）

### 🎨 四态视觉退化机制

| 状态 | strength | 视觉表现 | 触发特效 |
| :--- | :---: | :--- | :--- |
| **clear 清晰** | `≥ 0.7` | 正常透明度，墨色衬线字 | 2.4s 周期轻微呼吸 |
| **blurry 模糊** | `0.4 ~ 0.7` | 透明度下降，墨色变淡 | 高斯模糊（sigma 0.5~3.5）+ 4Hz 抖动 ±1.2px |
| **garbled 错乱** | `0.1 ~ 0.4` | 字符被相似字符替换 | 红色左偏 + 蓝色右偏（RGB 故障风） |
| **disappeared 消失** | `< 0.1` | 透明，保留占位宽度 | 完全透明 |
